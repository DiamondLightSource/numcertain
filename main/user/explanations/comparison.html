Comparison of Uncertain Values Explained#

Currently, the uncertain data type provides logical comparison for equality (i.e. eq & neq). The implementations of such are detailed in the Implementations table below:

Comparison Implementations#

Operator

Condition

Equal (EQ)
\[a = b \land \sigma_a = \sigma_b\]

Not Equal (NE)
\[a \neq b \lor \sigma_a \neq \sigma_b\]
